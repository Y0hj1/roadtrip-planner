# Скриншоты для README

Добавьте сюда PNG-скриншоты приложения. В корневом README используются такие имена файлов:

| Файл | Что снять |
|------|-----------|
| `01-landing.png` | Главная страница (лендинг) — `/` |
| `02-routes-catalog.png` | Каталог маршрутов с фильтрами — `/routes` |
| `03-route-details.png` | Страница одного маршрута с блоками «Что посмотреть» и отзывы — `/routes/:id` |
| `04-manage-routes.png` | Страница управления маршрутами — `/manage` |

Рекомендуемая ширина: 1200–1400 px. После добавления файлов они автоматически отобразятся в README на GitHub.
